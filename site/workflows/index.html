    Workflows¶

    End-to-end procedures for lab alerts, biobank, and devices. See the data flow diagram and Where data lives for tools and paths. Data and sample requests are under Tasks.

    Workflow What it covers
    CDL alert workflow CRL / CDL lab files from visit to nurse–physician follow-up
    RDL alert workflow Radboud diagnostic lab (RDL)—raw export, visit log linkage, processing, and alert follow-up
    Multi-centre sample data workflow Radboud blood-tube file, RL processing, boxing, biobank submission
    Biosample request workflow Request intake through biobank issuing, delivery, and archiving
    Device data workflow Device outputs, storage, QC cadence, links to cleaning scripts
    Device setup for visit iPad and laptop setup for VU-AMS, Omron, Castor, Nellcor, and Tanita before a visit

    Other recurring work (mailbox, visit log, subject ID log, Linked2Trial screeners, summary numbers) is documented in Recurring study routines.
