    Controlled vocabulary (FAIR)¶

    Semantic interoperability for NMCB / PCNN: ontology-backed variable mappings and OMOP CDM alignment. No active follow-up currently; documented for handover. Supports the Interoperable and Reusable FAIR principles alongside study-level metadata (FAIR overview).

    Two project streams¶

    Stream Page Output
    1. Ontology harmonization (SNOMED CT, LOINC, NCIT, …) Ontology harmonization Machine-readable variable ↔ term pairs; PCNN codebook integration; expert review workflow
    2. OMOP CDM mapping OMOP CDM mapping Table-level mappings (e.g. PERSON); YAML + ETL in nmcb-codebook; training from OHDSI

    Both build on the canonical nmcb-codebook repository (linkml/ + omop/).

    Canonical code repository¶

    Path in nmcb-codebook Role
    linkml/metadata/nmcb_codebook_metadata.yaml Variable definitions, value sets
    linkml/schema/nmcb_codebook_schema.yaml LinkML schema (next-step formalization)
    linkml/scripts/export_excel.py Excel export merging metadata + OMOP mappings
    omop/metadata/tables/*.yaml OMOP mappings by CDM table
    omop/sql/omop_etl/ Example load scripts
