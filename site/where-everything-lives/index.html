    Folder design principle¶

    A good storage structure should separate:

    • raw data
    • processed data
    • scripts
    • logs
    • documentation
    • outgoing deliverables

    Minimum expectation¶

    Every critical workflow documented on this site should point back to an actual folder, system, or repository here. Procedure pages live under Workflows.
