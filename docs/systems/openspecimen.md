# OpenSpecimen

*Lifecycle stages: [Storage](../lifecycle/storage.md) (sample tracking); [Issuing](../lifecycle/issuing.md) (sample requests)*

OpenSpecimen is the biobank system used for tracking samples.

## Purpose

- Sample inventory and tracking
- Links samples to participants and visits
- Supports sample request workflows

## Handover priorities

Document:

- access and roles
- how sample data is exported or linked to other systems
- relationship to [Biobank](biobank.md) data flow and templates
- sample request workflow integration
